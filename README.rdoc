==Modifiche a jekyl per disattivare gli hard_break di redclot

jekyll-0.5.4/lib/jekyll/site.rb

la funzione textile cambia in:

  def textile(content)
    rc = RedCloth.new(content)
    rc.hard_breaks = false
    rc.to_html
  end



RedCloth-4.2.2/lib/redcloth/formatters/html.rb

la funzione br di redclot diventa (non c'e' ne bisogno):

  def br(opts)
     puts "oh!"
     return "\n"
    if hard_breaks == false
      "\n"
    else
      "<br />\n"
    end
  end
