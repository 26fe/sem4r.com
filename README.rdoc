== To modify jekyl to add 'hard_break' for RedCloth

Open file: jekyll-0.5.4/lib/jekyll/site.rb,
change the 'textile' function to:

  def textile(content)
    rc = RedCloth.new(content)
    rc.hard_breaks = false
    rc.to_html
  end
